Para poder correr este proyecto es necesario tener SWI-Prolog instalado
Una vez instalado se deben incluir las siguientes rutas a la variable de entorno PATH
- C:\Archivos de programa\swipl\bin
- C:\Archivos de programa\swipl\lib\jpl.jar

Y finalmente incluir la libreria/modulo jpl en el proyecto a traves de tu IDE. Dicha libreria se encuentra en la ruta:
C:\Archivos de programa\swipl\lib\jpl.jar

Notas:
- El proyecto se desarrollo originalmente sobre el IDE IntelliJ IDEA Community Edition 2020.2.1
